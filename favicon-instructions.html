How to Create a favicon.ico for TV DAO

Option 1: Use an Online Favicon Generator

  1. Go to Favicon.io Converter
  2. Upload your tvdao-logo.png file
  3. Click "Download" to get the favicon package
  4. Extract the package and copy the favicon.ico file to your public folder

Option 2: Use RealFaviconGenerator

  1. Go to RealFaviconGenerator
  2. Upload your tvdao-logo.png file
  3. Customize the settings as needed
  4. Click "Generate your Favicons and HTML code"
  5. Download the favicon package
  6. Extract the package and copy the favicon.ico file to your public folder

After Creating the Favicon

Once you have the favicon.ico file:

  1. Place the favicon.ico file in your public folder
  2. Make sure the file name is exactly favicon.ico
  3. The metadata in layout.tsx has already been updated to use this file
  4. Restart your development server to see the changes
